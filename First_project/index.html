Super Mario Bros 3

Super Mario Bros 3 is one of the numerous retro games that earned its place in the hearts of many, this is a game made by Nintendo and the fourth game released by the director Shigeru Miyamoto in 1988. This game was Published for the NES console.

What is this game about?

This is a platform game about a plumber who has to rescue a princess from a dragon. he has to go through different worlds, defeating different enemies, and using exciting power-ups that make the game so much fun.

Playable Characters

  • Mario
  • Green Mario
